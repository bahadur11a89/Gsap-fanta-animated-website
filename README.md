# 🍊 GSAP Fanta Animated Website

## Fanta Landing Page Animation

This project is a **modern animated landing page inspired by the Fanta brand**.
It is built using **HTML5, CSS3, JavaScript, and GSAP (GreenSock Animation Platform)** to create smooth and engaging web animations.

The project demonstrates how animations can enhance **user experience and visual interaction** on modern websites.

## 🚀 Live Demo

https://fantabyrajbahadur.netlify.app/

## ✨ Features

* Smooth GSAP animations
* Interactive landing page design
* Modern and clean UI
* Scroll-based animation effects
* Responsive layout

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP Animation Library

## 🎯 Purpose of the Project

The purpose of this project is to practice **frontend animation techniques** and learn how to build **interactive landing pages using GSAP**.
